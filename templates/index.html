Smart Password Detection

Each component below describes one place a password might appear. In the context box, type the words you expect to see and drop a <{(password_value)}> marker where the password sits. A line break in the box means the password may be on the next line in the email. Spacing is flexible. Leave Password pattern blank for a generic match (any run of non-space characters), or set a regex to be specific. Matching is case-insensitive. Scanning is manual — close this and hit 🔍 Scan.

Suspected Customers List

Map a keyword to a customer organization. When a mail's content (subject or body) matches the keyword, the mail resolves to that organization — used for the download file name/metadata and the exported report's customer-organization column. The first matching keyword (top to bottom) wins. The keyword is a full search expression, so one mapping can need several terms.
